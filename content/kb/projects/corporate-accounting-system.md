## Overview

Enterprise accounting and business management system developed using Laravel, Vue.js, and Microsoft SQL Server.

Implemented and customized for multiple companies across different industries, supporting daily accounting operations, financial reporting, inventory management, purchasing, sales, and administrative workflows.

## Key features

General ledger management. Accounts receivable and accounts payable. Sales and purchasing workflows. Inventory tracking and stock movement monitoring. Financial reporting and business analytics. User permissions and role-based access control. Multi-company support and configurable business processes.

## Responsibilities

Developed new features, maintained existing modules, fixed production issues, implemented client-specific customizations, and supported system deployments.

Worked directly with databases, business logic, reporting requirements, and user workflow improvements.

## Technical experience gained

Large-scale Laravel application architecture. Vue.js frontend development. Microsoft SQL Server database management. Stored procedures and query optimization. Production troubleshooting and maintenance. Client requirement analysis and implementation.

## Infrastructure and deployment

Participated in application deployment, server configuration, environment setup, production support, and ongoing system maintenance. Gained hands-on experience with software delivery, deployment workflows, and operational support.
