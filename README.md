## [ ABOUT ]

I build custom accounting systems, LMS platforms, and AI-enhanced web applications — with a focus on clean architecture, secure deployments, and software that actually survives contact with real users. Most of my work lives behind the scenes: enrollment systems, payment flows, video pipelines, approval workflows. This site is one of the few things I've built that's meant to be looked at directly.

## [ WHAT THIS SITE DOES ]

- **Terminal Secretary** — an AI concierge (Gemini) that boots up like a real terminal and answers questions about my work, in character the whole way through.
- **Live status, not stats** — a running clock in Manila time, my latest verifiable public GitHub activity, and a manually-written "currently building" line. Nothing here is a number I graded myself.
- **A real contribution graph** — pulled live from GitHub, not a static image pretending to be current.
- **Categorized skills** — grouped honestly, with a real brand icon where one exists, and a plain marker where it doesn't, instead of forcing a logo that isn't real.

## [ DESIGN NOTES ]

Dark ink background, sage and gold accents, hairline rules standing in for a ruled notebook. Fraunces carries the display type; IBM Plex Mono carries everything that behaves like a system reading itself back to you. The whole site runs on one idea: nothing here states a number about me that I decided for myself. Every claim is either live, linked, or written in the first person as a status — not a score.
